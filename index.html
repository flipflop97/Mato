Mato

Mato is a modern-looking icon pack for Linux. Inspired by Googles Material Design guidelines, these icons are made to look simple and colorful with subtle use of shadows. The icons are based on their original icons so they stay recognizable while gaining this new look.

All currently available icons are displayed below. If you'd like to submit icon requests or bug reports, head over to the GitHub issue tracker.

Installation

Arch Linux users can install Mato using the AUR package mato-icons.

Ubuntu and Debian builds are currently still under development.

Packages for Fedora, openSUSE, CentOS and some other distributions are available here.

If your current distribution isn't supported, you can download and install it manually from the releases page or by installing the development version using the command below.

sh -c 'mkdir -p ~/.icons && rm -rf ~/.icons/Mato/ && git clone https://github.com/flipflop97/Mato.git ~/.icons/Mato/'
